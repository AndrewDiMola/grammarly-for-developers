Suggestion Categories (Default)

When a writing suggestion category is set to "off", as is the case with the seven categories of suggestions that are muted with default settings, any suggestions belonging to that category will be muted (not visible to the user); however, if a user has connected their Grammarly account to your application, the full set of customized suggestions from their user profile will always take precedence (overriding these configurations).

Default settings for Suggestion Categories

By default, the Grammarly Text Editor SDK mutes (turns "off") seven categories of writing suggestions.

  • Conjunctions at the start of a sentence: Suggestions for using conjunctions such as "but" and "and" at the beginning of sentences ("conjunctionAtStartOfSentence").
  • Prepositions at the end of a sentence: Suggestions for using prepositions such as "with" and "in" at the end of sentences ("prepositionAtTheEndOfSentence").
  • Informal academic pronouns: Suggestions for using personal pronouns such as "I" and "you" in academic writing ("informalPronounsAcademic").
  • The Oxford comma: Suggestions for adding the Oxford comma after the second-to-last item in a list of things. ("oxfordComma").
  • The passive voice: Suggestions regarding use of the passive voice ("passiveVoice").
  • Unnecessary ellipsis: Suggestions regarding unnecessary use of ellipses (...) ("unnecessaryEllipses").
  • Stylistic fragments: Suggestions for completing all incomplete sentences, including stylistic sentence fragments that may be intentional ("stylisticFragments").
